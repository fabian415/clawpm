VERSION: v1.0.2
DATE: 2026-05-22
----------------------------------------------------------------
[錯誤修正]
- 修正 Azure gpt-5.4 在聊天過程中停止回應的問題
  解法：
  (1) 開啟新的 session 以重置對話狀態
  (2) 延長 OpenClaw LLM timeout 至永久（預設為 2 分鐘）
  (3) 啟用 tokenjuice plugin，自動壓縮工具輸出，減少 context 累積速度

[新功能]
- 新增 Danger Zone 功能，允許 Admin 直接刪除整個團隊（含容器與所有帳號）
- 刪除團隊按鈕限 Admin 角色可見，一般 User 無法操作
- 完成會話紀錄（Chat Session）功能
